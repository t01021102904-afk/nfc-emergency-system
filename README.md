This project aims to protect seniors, veterans, and anyone who may face a medical emergency by using **NFC/QR-based Emergency Cards**.

When the card is scanned:

- Automatic SMS alert sent to guardian  
- Real-time GPS location sent to server  
- Role verification (EMT, Hospital, Police, Civilian)  
- Essential medical info displayed for authorized responders  
- Anonymous calling system (WebRTC or Proxy)  
- Admin dashboard for card registration/deactivation/loss reports  
- Guardian dashboard with maps + scan history  

We are recruiting volunteer developers to help build this life-saving system.
